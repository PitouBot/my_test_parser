# 🕷️ Учебные парсеры для parsinger.ru

Два парсера для сбора информации о товарах с тестового(учебного) сайта [parsinger.ru](https://parsinger.ru/html/).

## 📦 Что внутри

- **`parser_csv.py`** — собирает основные данные о товарах и сохраняет в CSV
- **`parser_json.py`** — собирает расширенные данные (с описанием по параметрам) и сохраняет в JSON
